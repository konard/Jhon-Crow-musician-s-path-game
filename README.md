# Musician's Path Game

A static MVP for a gamified musician release checklist. The app turns release-growth actions into points, timeline acceleration, and a persistent local checklist.

## Run locally

Open `index.html` in a browser, or serve the directory with any static file server.

## Test

```bash
npm test
```

## Deployment

The repository includes a GitHub Pages workflow in `.github/workflows/pages.yml`. It runs tests on the pull request branch and deploys the static site from `main` after merge.
